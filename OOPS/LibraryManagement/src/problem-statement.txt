🎯 Problem: Library Management System (Console-Based)

Build a Java console application with the following features:

📌 Functional Requirements:
Add a book
View all books
Issue a book
Return a book
Exit
📌 Book Details:

Each book should have:

Book ID
Title
Author
Status (Issued / Available)
📌 Constraints:
Follow proper OOP principles
Code must be modular (avoid putting everything in main)
Handle invalid operations (e.g., issuing an already issued book, returning a non-issued book)
